# 短线选股策略系统

2026年A股短线选股技能，基于**通达信本地日K线数据**，计算真实技术指标（MACD/MA/RSI），集成5大短线策略共振。

## 数据源

| 数据源 | 类型 | 用途 | 特点 |
|:---|:---|:---|:---|
| **通达信本地数据** | `.day`日K线文件（D:\new_tdx\vipdoc） | 主力选股 | 全量9989只，可算MACD/MA/RSI，30秒扫完 |
| **通达信行情直连** | pytdx直连（180.153.18.170:7709） | 盘中实时监控 | 23只实时行情，异动提醒 |

**用户每天收盘后在通达信下载盘后数据，系统自动使用最新数据。**

## 复盘结论（5/29选股→6/1验证）

选中的23只中，7只上涨、13只下跌、3只平盘。复盘发现了关键规律：

| 信号 | 说明 |
|:---|:---|
| **前5日涨幅>15%** | 短期获利盘太重，次交易日易回调 |
| **选股日涨>6% + 前5日>10%** | 高潮见顶信号，买入即套 |
| **前5日涨幅<10% + RSI<80** | 还有上涨空间，持有关注 |

已落实为选股过滤条件，避免再选出第二天就跌停的股票。

## 策略体系（基于真实技术指标）

### 策略1: 低位放量首板 🚀
- **条件**: 涨幅3-8% + 量比>1.8 + 均线多头 + MACD柱升高 + 均线角度≥30° + **突破20日平台**
- **对应**: 首板突破20日平台（三步伏击涨停笔记）

### 策略2: 连板接力弱转强 🔥
- **条件**: 10日内有涨停 + 今日上涨 + 均线多头
- **对应**: 连板接力、龙头二板

### 策略3: 趋势加速 ⚡
- **条件**: 均线多头 + 放量 + MACD多头/金叉
- **对应**: 主升浪加速

### 策略4: N字反包 🔄
- **条件**: 10日内涨停 → 回调 → 今日再涨
- **对应**: 游龙戏凤、第二波拉升

### 策略5: 多维精选 💎
- **条件**: MACD多头 + 均线多头 + 放量
- **对应**: 三维度共振

**综合排名**: 按策略共振数降序 + 涨幅降序，≥2个策略进入推荐池
